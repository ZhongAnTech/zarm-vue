### API

#### Badge Attributes

| 属性      | 类型           | 默认值   | 可选值／参数                                        | 说明         |
| :-------- | :------------- | :------- | :-------------------------------------------------- | :----------- |
| theme     | string         | 'error'  | 'default', 'primary', 'success', 'warning', 'error' | 主题         |
| shape     | string         |          | 'dot', 'rect', 'radius', 'round', 'circle', 'leaf'  | 形状         |
| text      | string, number |          |                                                     | 显示文字     |
| sup       | boolean           | false    |                                                     | 是否上标位置 |

#### Badge Events

| 事件名称 | 说明               | 回调参数       |
| :------- | :----------------- | :------------- |
| click    | click 时触发的事件 | event 事件对象 |
